# Linux Shell Script

使用 .sh 副檔名來命名 Shell Script 檔案。然後將該檔案設定為可執行
```
chmod +x demo.sh
```
- 變數宣告 \
variable1=value
若是值內有空白則需要使用 '' 或 "" 包裹起來
 
使用變數方式為 ${變數名稱} ex:echo ${value} \
刪除變數使用unset

- 運算式 \
expr命令可以用來做數值運算、數值或字串比較、字串匹配、字串提取、字串長度計算等功能。 它還具有幾個特殊功能,判斷變數或參數是否為整數、是否為空、是否為0等。
```
$expr 10 + 10
20
$expr 1500 + 900
2500
$expr 30 / 3
10
$expr 30 / 3 / 2
5
```
